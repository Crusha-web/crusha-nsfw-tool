📱 スマホ24時間アクセス設定

PCをシャットダウンしてもスマホからアクセスできるよう、GitHub Pagesにデプロイします。以下の手順で一度だけ設定すれば永久に使えます。

1
GitHubにリポジトリ作成
github.com → New repository → 名前をcrusha-studioに設定 → Public → Create
2
ファイルをアップロード
このフォルダの全HTML + manifest.json + sw.js を「Add file → Upload files」でドラッグ&ドロップ
3
GitHub Pages有効化
Settings → Pages → Branch: main → /root → Save
4
スマホからアクセス
スマホブラウザで下記URLを開き、「ホーム画面に追加」でアプリとしてインストール完了
🌐 https://[あなたのGitHubユーザー名].github.io/crusha-studio/
