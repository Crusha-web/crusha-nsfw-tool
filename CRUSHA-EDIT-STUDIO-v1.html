📋よく使うFFmpegコマンド集
🎬 動画結合 (concatファイル方式)
ffmpeg -f concat -safe 0 -i filelist.txt -c copy output.mp4
🎵 BGMミックス (音量調整付き)
ffmpeg -i video.mp4 -i bgm.mp3 -filter_complex "[1:a]volume=0.7[bgm];[0:a][bgm]amix=inputs=2:duration=first" -c:v copy output.mp4
🌊 クロスフェードトランジション
ffmpeg -i clip1.mp4 -i clip2.mp4 -filter_complex "[0:v][1:v]xfade=transition=dissolve:duration=0.5:offset=2.5[v]" -map "[v]" output.mp4
📐 解像度変換 + カラーグレーディング
ffmpeg -i input.mp4 -vf "scale=1920:1080,eq=brightness=0.05:contrast=1.2:saturation=1.3" -c:v libx264 -crf 18 output.mp4
📜 字幕焼き込み (ASS形式)
ffmpeg -i video.mp4 -vf "ass=subtitles.ass" -c:v libx264 -crf 18 output_with_subs.mp4

🎬DaVinci Resolve 作業手順
① プロジェクト作成
DaVinci Resolve起動 → New Project → プロジェクト名: CRUSHA_LIVE_2026
Settings → Master Settings → 解像度: 1920×1080 / フレームレート: 24fps
② メディアのインポート
Media Pool → Import Media → 生成した全mp4ファイルをドラッグ&ドロップ
BGM音声ファイル(wav/mp3)も同様にインポート
③ EDLインポート
File → Import → Timeline → 出力したEDLファイルを選択
タイムラインに自動配置される（パスが一致しない場合は手動で再リンク）
④ カラーグレーディング
Color ページ → Nodes → Serial Node追加 → 各クリップにLUT適用
CRUSHA推奨: 彩度+20、コントラスト+0.15、ハイライトシアン寄り
⑤ 書き出し
Deliver ページ → Render Settings → YouTube/Vimeo or Custom → CRUSHA EXPORT STUDIOで設定値確認 → Render
